# Faktoro

Faktoro is a multiplatform mobile app for time tracking, invoicing, and client management.

## Features

- Time tracking with running timers, pause/resume, and timesheets
- Invoice generation with PDF and XML export
- Client management, price lists, and custom rates
- Offline-first data storage
- Sync between devices

## Tech Stack

- React Native with Expo
- Expo Router
- WatermelonDB
- SQLite

## Development

Install dependencies:

```bash
npm install
```

Start the app:

```bash
npx expo start
```

Run on Android or iOS:

```bash
npm run android
npm run ios
```
